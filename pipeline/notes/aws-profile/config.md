## PROFILE 

[How to set up and work with AWS CLI profiles](https://www.youtube.com/watch?v=ptbxhz7TWmE)

### Config
```
[default]
region = us-east-1
output = json

[profile dev]
region = us-east-1
output = json

[profile qa]
region = us-east-1
output = json

[profile prod]
region = us-east-1
output = json
```
